it's about time.

_Friday 17th January 2020._

---

all this talk +
about healing +
and learning +

in the new year +

finally makes sense. +

-

it's been just +
over two weeks +

since the new year +

but now, +

it's clear +

this healing +
and learning +

I previously spoke of; +

it has only +
just started. +

-

there are a lot +
of complex things +

to work through +
which naturally +

require a lot +
of time, effort +
and hard work +

but first +

before any +
of this, +

I need to get +
some sleep...
