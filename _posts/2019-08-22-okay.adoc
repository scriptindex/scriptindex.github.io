= okay
:hp-tags: poetry
:published-at: 2019-08-22

always.

---

we can get +
in our own way +

but we will try to +
make it through. +

people will try +
to get their way +

but we will surely +
make it through. +

life will try +
its best to +
test us +

but we will always +
make it through. +

there is +
no need +
to worry. +

everything is +
going to be +
okay.
