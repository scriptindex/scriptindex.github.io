= Link On Line
:hp-tags: poetry
:published-at: 2018-03-13

Early morning vibes.

---

when there +
is no one +

not a single +
individual +
what is to stop +
you from taking +
it all +

droplets merging +
thick black tar +
merging with +
flat merging +
merging to you +
when I not +
merging there +
now +

reality +
merging +
back into +

dust
