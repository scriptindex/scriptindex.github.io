= Drift Focus
:hp-tags: poetry
:published-at: 2017-04-26

Catching up on the +
old pleasures of life.

---

Tropic nightlife +
tranquil backdrop +

surfing through +
the radio waves +

Interior reality +
keeps distance +
to a minimum +

as these +
vibrant sights +
bleed each other
